Описание данных Инициативы G DATA по борьбе с вредоносными программами для Internet Security

Что такое Инициатива G DATA по борьбе с вредоносными программами?
Специалисты антивирусной лаборатории G DATA исследуют способы защиты клиентов компании G DATA от вредоносных программ. Чем больше информации о таких программах, тем эффективнее разрабатываемые механизмы защиты. Чтобы собрать сведения для анализа, была создана Инициатива G DATA по борьбе с вредоносными программами. Своим участием вы способствуете более безопасному использованию смартфонов и других мобильных устройств всеми клиентами G DATA.

Какие данные собираются?
При обнаружении подозрительного исполняемого файла возможен сбор следующей информации:
-  Версия протокола Инициативы по борьбе с вредоносными программами
-  Номер версии программы G DATA, а также используемого движка
-  Номер версии используемых сигнатур
-  Индивидуальная для устройств информация об операционной системе, языковых установках (локальных) и GUID
-  Информация об используемом на момент идентификации провайдере
-  Индивидуальная для устройств информация о производителе, модели и версии устройства
-  Наименование и ID сигнатуры для каждого найденного вредоносного файла
-  Информация о подозрительном файле:
  *  Контрольная сумма файла или его подозрительных частей
  *  Дата и время обнаружения вредоносного ПО
  *  Используемый провайдер
  *  Место, откуда был загружен вредоносный файл
  *  Проявления вредоносного файла
  *  Имя файла и стандартизированный путь (имя пользователя заменяется на %USER%)

При тарифах, рассчитываемых по объему данных, могут возникать дополнительные издержки. Объем данных - это несколько килобайт. Как правило, это стоит совсем недорого.

ВАЖНО: При этом не осуществляется отправка личных данных. Переданные сведения невозможно использовать для идентификации пользователя смартфона.

Как используются полученные данные?
При обработке и сохранении данных мы учитываем законодательные требования по защите данных и разглашению информации в соответствующих странах. G DATA прилагает огромные усилия для защиты всех данных от несанкционированного доступа.

Оценка данных осуществляется в G DATA Security Labs и используется для получения ответов на вопросы исследований из области безопасности информационных технологий. Важнейшей целью является исследование рисков для системы безопасности и разработка механизмов защиты. Среди примеров применения указанных данных можно назвать, например, создание списков для блокировки, статистические данные для публикации в специализированных статьях, а также разработку нормативных документов для сферы защиты данных.
